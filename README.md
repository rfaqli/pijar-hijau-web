Platform Pusat Informasi dan Jejaring Karier Hijau.

## Project Purpose

This project aims to provide an integrated platform for individuals and organizations interested in Indonesia's green industry sector. It connects users with career opportunities, quality education, and sustainable investment options to support the transition to a greener economy.

The website is designed to be:
- Easy to navigate and user-friendly
- Visually appealing with a modern green tech motif
- Informative and comprehensive for various user needs

## Target Users

- Professionals seeking careers in the green industry
- Students and individuals wanting to learn about sustainable practices
- Investors looking for sustainable and profitable green projects

## Core Pillars

- **Peluang Karir** – Find job vacancies in the rapidly growing green industry sector.
- **Pendidikan Berkualitas** – Learn green industry skills from experts with standardized certifications.
- **Investasi Berkelanjutan** – Realize a green future with smart and profitable investments.

## Main Features

- Landing page with clear value propositions and responsive design
- Career portal mapping (Peluang Karir)
- Educational classes and certification info (Kelas & Sertifikasi)
- Investment opportunities overview (Investasi)
- News and green industry updates (Berita)
- Contact information (Kontak)

## Technologies Used

- **React** – UI library
- **TypeScript** – Type-safe JavaScript
- **Tailwind CSS** – Utility-first CSS framework for styling
- **Vite** – Fast frontend build tool
- **Lucide React** – Clean and consistent icons
